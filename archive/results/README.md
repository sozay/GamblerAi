# Backtest Results Archive

This directory contains historical backtest result charts and outputs.

## Chart Directories

### backtest_charts/
Initial backtest visualization outputs:
- `equity_curve.png` - Portfolio equity over time
- `performance_by_symbol.png` - Per-symbol performance comparison
- `win_loss_distribution.png` - Win/loss distribution histogram

### backtest_charts_final/
Final backtest run outputs with same chart types.

## Notes

- These are archived results from historical backtest runs
- Charts are generated by `scripts/visualize_backtest.py`
- For current backtest results, see `docs/results/`
- New chart outputs will be generated in project root during backtests
